# VancouverPianoIndex_

This is a simple, WIP website documenting free and accessible pianos across Vancouver. Users can explore piano locations, view access notes, and open each location directly in Google Maps.

Features:
- Interactive leaflet map
- Clickable piano titles linking to Google Maps
- Brief notes on the accesibility and number of pianos
- Piano data stored and loaded dynamically from Firebase Firestore

Technologies Used:
- HTML / CSS / JavaScript
- Leaflet.js
- Firebase Firestore 
- Font Awesome
